# Gian Schneider Portfolio

Dieses Repository enthält die persönliche Portfolio-Website von Gian Schneider.

## Zweck

Die Website präsentiert kurz Projekte aus dem Geomatik-Studium an der FHNW und zeigt grundlegende berufliche Informationen zu Gian Schneider.

## GitHub Pages

Das Repository ist für GitHub Pages geeignet. Nach dem Push auf GitHub kann die Seite über den Branch „main“ oder „master“ als GitHub Pages-Website veröffentlicht werden.
